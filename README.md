### Usage:
```sh
$ ruby deobfuscate.rb [FILE] [POWER]
```
ex.
```sh
$ ruby deobfuscate.rb NoCombatHacks.txt 70
```

To bruteforce the power use
```sh
$ ruby deobfuscate.rb [FILE]
```
